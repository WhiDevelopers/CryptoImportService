# CryptoImportService

Webapp that imports crypto data to database

Install node & npm

https://nodejs.org/en/

Install npm dependecies
```
cd CryptoImporter
npm install
```

Run in CryptoImporter folder
```
npm run dev
```

Tevo 1 liner. Selleks et terminal lahti teha ja kopeerida 1 lause
```
cd cryptoimporter && npm run dev
```
